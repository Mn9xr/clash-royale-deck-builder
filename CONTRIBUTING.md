# Contributing

Thanks for checking out this project.

This started as a fun side project, but contributions, suggestions, and improvements are still welcome.

## Ways to contribute

You can help by:

- fixing bugs
- improving the UI
- improving deck logic
- improving chatbot/AI behavior
- cleaning up code
- improving documentation
- suggesting features
- reporting issues

## Before contributing

Please:

1. read the `README.md`
2. check existing issues before opening a new one
3. keep changes focused and clear
4. avoid unrelated large rewrites unless discussed first
