# Code of Conduct

## Our Pledge

We want this project to be a welcoming, respectful, and constructive space for everyone.

As contributors, maintainers, and participants, we pledge to make participation in this project a harassment-free experience for all, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that helps create a positive environment include:

- being respectful and constructive
- giving helpful feedback
- being open to different ideas and approaches
- showing patience, especially with beginners
- focusing on improving the project
- communicating clearly and professionally

Examples of unacceptable behavior include:

- harassment, bullying, or discrimination
- insulting, demeaning, or aggressive comments
- trolling or intentionally disruptive behavior
- personal attacks
- hateful or exclusionary language
- doxxing or sharing private information without permission
- spam or repeated unwanted self-promotion

## Project Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- pull requests
- discussions
- comments
- wiki content
- any other official project communication spaces

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may take appropriate action in response to behavior they consider inappropriate, threatening, offensive, or harmful.

Possible actions may include:

- warning the person involved
- removing comments or content
- temporarily limiting participation
- banning repeat or serious offenders from the project space

## Reporting

If you experience or witness unacceptable behavior, please report it privately to the project maintainer.

When reporting, include:
- what happened
- where it happened
- screenshots or links if helpful
- any other useful context

All reports will be reviewed and handled as fairly as possible.

## Enforcement Guidelines

Maintainers may consider:
- the severity of the behavior
- whether it was repeated
- whether it was intentional
- the effect it had on others
- whether a warning or stronger action is appropriate

## Attribution

This Code of Conduct is inspired by common open source community standards, including the Contributor Covenant, but simplified to fit this project.
